## 提示词模板
提示词功能根据不同的提示词，组合成更复杂全面的提示词
如：项目规范提示词，项目注释提示词，工具类提示词等
## 功能节点编排
可以将多个基础功能进行编排，形成可以像工作流那样，按节点执行
### 基础功能有如下：
#### 1. 鼠标选中内容 （获取鼠标选中的内容）
#### 2. 读取鼠标选中文件内容 （获取鼠标选中的文件的文件内容）
#### 3. 提示词编辑（编辑提示词）
#### 4. 正则过滤处理（正则过滤处理输入）
#### 5. 大模型处理（调用大模型）
#### 6. 根据类名获取类内容（通过idea的类查找功能，通过类名查找对应的java类内容，只支持java文件）

### 如：读取文件-> 正则过滤文件内容->组合形成提示词->选择大模型执行
