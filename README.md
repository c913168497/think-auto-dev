ThinkAutoDev是基于AutoDev进行的二次开发
该插件着重于提示词模板，功能节点编排
安装：

提示词模板可以根据不同的提示词，组合成更复杂全面的提示词
如：项目规范提示词，项目注释提示词，工具类提示词等
功能节点编排
可以讲多个基础功能进行编排，形成可以像工作流那样，按节点执行
如：读取文件-> 正则过滤文件内容->组合形成提示词->选择大模型执行
